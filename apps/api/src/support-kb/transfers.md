# Transfers

## How to send a transfer?
Navigate to Inventory > Transfers > New Transfer. Select the source and destination channels, add the items and quantities. Once saved, the status becomes AWAITING_RECEIVER. Stock is 'in transit' and not available at either location.

## Receiving a Transfer
The receiving branch must inspect the items and click 'Receive Transfer'. If quantities match, stock is added to their local inventory. If there is a mismatch, use the 'Dispute' option.

## Transfer Disputes
If 10 items were sent but only 9 arrived, the receiver should dispute the line. This requires a reason. The system creates a 'Shrinkage Expense' automatically to account for the lost value, which goes into the ledger for audit.

## Transfer Format
Transfer numbers follow the TRX-XXXX format.
